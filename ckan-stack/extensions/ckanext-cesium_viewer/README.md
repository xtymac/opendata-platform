# ckanext-cesium-viewer

This extension provides a lightweight CKAN resource view that embeds a CesiumJS viewer for 3D Tiles resources or wraps external Cesium web map packages generated by 3DCityDB.

## Features

- Automatically renders resources ending in `.html` inside an iframe.
- For CityGML exports exposing a `tileset.json`, the view spins up a basic CesiumJS scene directly in CKAN.
- Optional `tileset_url` override per resource view configuration.
- Remote or CKAN-hosted ZIP archives containing 3D Tiles can be downloaded, extracted, and served automatically from the shared `/3d/auto/<resource_id>/` path. Supply `tileset_path` if the archive contains more than one tileset.

After installing, add `cesium_viewer` to `ckan.plugins` and create a resource view for any dataset resource referencing your 3D map or 3D Tiles.
